PRACTICA 1 - IMPLEMENTACIO DFA 

--------------------------Project Overview-------------------------

Este proyecto consiste en la implementación de un Autómata Finito Determinista (DFA) en un entorno 
Java basado en ANTLR4. El objetivo principal es el desarrollo de la clase DFA y los distintos autómatas 
que serán testeados para comprobar su funcionamiento, permitiendo así construir autómatas a partir de especificaciones 
formales y probar diversas cadenas para comprobar si son aceptadas o no.

El programa lee archivos de texto los cuales contienen la especificación de distintos autómatas y su estructura y luego 
procesa múltiples cadenas de entradas. En caso de que la función de transición no esté definida por completo se asumirá la existencia 
de un nodo trampa.

Por último, este trabajo contiene herramientas de prueba a modo de tests que sirven para verificar el correcto funcionamiento 
de la implementación, los cuales ya vienen predefinidos y con la opción de ampliarlos y añadir nuevas especificaciones.
